# Blackbriar

If you need to map some JSON data to other JSON data arbitrarily then you can use Blackbriar

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'blackbriar'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install blackbriar
